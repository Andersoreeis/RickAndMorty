Este projeto é uma homenagem à série de animação de comédia e ficção científica, Rick and Morty

inclui informações detalhadas sobre os personagens e recursos da série.

Para reunir essas informações, usei uma API pública conhecida como The Rick and Morty API

construí este site com

  • html
  • CSS
  • JAVASCRIPT
  • API PÚBLICA

Na página inicial, você encontrará informações sobre a série e a API, bem como a opção de pesquisar seus personagens favoritos pelo nome. Já na página de personagens, você pode ver uma lista completa de todos os personagens apresentados na série. Para obter mais informações sobre um personagem específico, basta clicar em sua imagem e uma página detalhada com informações sobre o personagem será exibida. criei uma páginação personalizada para garantir que a navegação pelo site fosse fácil e intuitiva.

Espero que goste do resultado final!

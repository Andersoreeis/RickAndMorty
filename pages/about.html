Hello world

This project is a tribute to the comedy and science fiction animated series, Rick and Morty.

It includes detailed information about the characters and features of the series.

To gather this information, I used a public API known as The Rick and Morty API.

I built this website using:

  • HTML
  • CSS
  • JAVASCRIPT
  • PUBLIC API

On the home page, you'll find information about the series and the API, as well as the option to search for your favorite characters by name. On the characters page, you can see a complete list of all the characters featured in the series. To get more information about a specific character, simply click on their image and a detailed page with information about the character will be displayed. I created a custom pagination to ensure that navigating through the site was easy and intuitive.

I hope you enjoy the end result!
